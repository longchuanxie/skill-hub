# SkillHub 技术选型澄清与开发计划

## 1. 技术选型澄清

### 1.1 后端技术栈

| 层级 | 技术选型 | 版本要求 | 说明 |
|------|----------|----------|------|
| 运行时 | Node.js | 20.x LTS | JavaScript运行环境 |
| 框架 | Express.js | 4.x | Web应用框架 |
| 数据库 | MongoDB | 7.x | 文档型数据库 |
| ODM | Mongoose | 8.x | MongoDB对象建模 |
| 认证 | JWT | jsonwebtoken 11.x | JSON Web Token |
| 密码加密 | bcryptjs | 2.x | 密码哈希 |
| 验证 | express-validator | 7.x | 请求验证 |
| 文件存储 | 本地存储 / MinIO | - | Skill文件存储 |
| 内容检测 | 自定义检测服务 | - | 提示词安全检测 |
| 缓存 | Redis | 7.x | Token缓存、会话缓存 |
| 日志 | winston + morgan | - | 应用日志 |
| API文档 | swagger-jsdoc + swagger-ui-express | - | 自动生成API文档 |

**替代方案说明：**

- **数据库**: 暂不使用PostgreSQL，MongoDB的文档结构更适合Skill和Prompt的灵活JSON存储
- **缓存**: Redis为可选，生产环境建议部署，开发环境可简化
- **文件存储**: 初期使用本地存储，企业部署可切换MinIO或S3兼容存储

### 1.2 前端技术栈

| 层级 | 技术选型 | 版本要求 | 说明 |
|------|----------|----------|------|
| 框架 | React | 18.x | 前端UI框架 |
| 语言 | TypeScript | 5.x | 类型安全 |
| 构建工具 | Vite | 5.x | 快速构建工具 |
| UI组件库 | animate-ui | latest | 动画UI组件库 |
| 样式方案 | Tailwind CSS | 3.x | 原子化CSS框架 |
| 状态管理 | Zustand | 4.x | 轻量级状态管理 |
| 路由 | React Router | 6.x | 前端路由 |
| HTTP客户端 | Axios | 1.x | HTTP请求库 |
| 表单处理 | React Hook Form | 7.x | 表单管理 |
| 数据校验 | Zod | 3.x | Schema校验 |
| 测试 | Vitest + React Testing Library | - | 单元测试和组件测试 |
| E2E测试 | Playwright | - | 端到端测试 |

**替代方案说明：**

- **状态管理**: 不使用Redux，Zustand更轻量且API简洁
- **样式**: 不使用styled-components，Tailwind CSS更高效
- **UI库**: animate-ui为主，可根据需要补充其他组件
